inlinestatic

Set the shared IRQ state structure.

This function sets whether the master or slave driver IRQ handler will be invoked and to set the pointer to the driver run-time state structure associated with the desired module instance number. This is not a public API.

void dspi_slave_irq_handler ( void *  state)

DSPI Slave Generic IRQ handler.

This handler uses the callbacks stored in the dspi_slave_state_t struct to transfer data either from the data source or to the data sink functions. This is not a public API as it is called whenever an interrupt occurs.
