Data Structure Documentation

struct ring_buffer_t

Ring buffer object.

The read and write heads will point to the same byte either when the ring buffer is empty, or when it is full. You can use the filled member to tell which of those two cases applies.

Data Fields
uint8_t * buffer Pointer to the buffer.
uint32_t filled Number of bytes currently in the buffer.
uint32_t length Length in bytes of the buffer.
uint8_t * readHead Pointer to next byte to read.
uint8_t * writeHead Pointer to next byte to write.
